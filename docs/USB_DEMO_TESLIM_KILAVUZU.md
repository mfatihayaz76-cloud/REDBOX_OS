# REDBOX OS Mac Demo USB Teslim Kılavuzu

Sürüm: 1.1.0
Build: 2

## USB teslim içeriği

USB'de `REDBOX_OS_MAC_DEMO_1.1.0_BUILD_2` klasörü bulunmalıdır:

- `REDBOX_OS-1.1.0-2.dmg`
- `REDBOX_OS-1.1.0-2-manifest.json`
- `SHA256SUMS.txt`
- `Belgeler/` klasörü
- `BASLANGIC.md`

Özel lisans anahtarı, gerçek müşteri RBX1 dosyası, canlı veritabanı,
yedek veritabanı veya özel işletme verisi USB'ye konulmaz.

## Kurulum

1. USB klasörünü Mac'e kopyalayın.
2. SHA-256 değerini manifest ile karşılaştırın.
3. DMG'yi açıp uygulamayı Applications klasörüne sürükleyin.
4. İlk kurulumda `DEMO` seçin.
5. Firma/tesis için demo bilgileri ve yönetici hesabını oluşturun.
6. Kod girmeden 30 günlük demoyu kullanın.

## Ticari geçiş

Müşteri satın aldığında aynı cihazdan lisans talebi dışa aktarılır.
Yetkili otoritenin ürettiği cihaza bağlı RBX1 lisansı Lisans
Merkezi'nde etkinleştirilir.
